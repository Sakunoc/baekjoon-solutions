https://www.acmicpc.net/problem/8472

유형 : MCMF

풀이 : 

그래프 모델링 - 1개의 소스, 3 * N개의 노드, 1개의 싱크로 구성되는 그래프 배열 생성
연속된 N일에서 K개 만큼만 선택되어야 함 -> 소스를 에서 첫번째 노드로 가는 유량을 K로 설정, 비용은 0
노드를 선택했을 경우의 간선을 추가해야 함.

  ---N--> (N = 3)
s n n n n n n n n t

한번 선택한다면 N만큼의 범위동안 선택할 수 있는 갯수, 즉 유량이 줄어야하고. 범위를 벗어난다면 선택할 수 있는
갯수, 유량이 늘어나야 함.
따라서 모든 노드에 대해서 자신을 시작으로 N만큼 건너뛴 곳을 끝으로 간선을 연결함(유량 : 1, 비용은 그 날짜의 온도 * -1)

만약 N만큼 건너뛴 곳이 싱크를 지난다면 싱크로 향하게 고정

for i, x in enumerate(list(map(int, input().split())), start = 1): # node jump
    jump = i + N

    if jump >= max_length:
        add_edge(i, max_length - 1, 1, -x)
    
    else:
        add_edge(i, jump, 1, -x)

그리고 이 날짜의 휴가를 지나치는 경우를 생각해
n -> n + 1로 가는 간선을 추가해야 함(유량 INF, 비용 0)

결과적으로, 그냥 지나치거나 휴가를 갈지를 선택하는 경우를 그래프로 나타내게 됨
